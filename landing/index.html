Antfarm
GitHub

Multi-agent workflow orchestration for OpenClaw. Define pipelines in YAML, let agents execute autonomously.

Install

Install GitHub.com/snarktank/antfarm

Tell your OpenClaw agent to install Antfarm. That's it.

Example

# Install a workflow
antfarm workflow install ./workflows/feature-dev

# Run a task
antfarm workflow run feature-dev "Ship the onboarding prompt"

# Check status
antfarm workflow status "Ship the onboarding prompt"

Dashboard

Task dashboard showing workflow runs and step status

Commands

Command Description
antfarm workflow install <source> Install a workflow from a local path, GitHub repo, or URL
antfarm workflow run <id> <task> Start a new run with the given task
antfarm workflow status <task> Check status of a running or completed task
antfarm workflow update <id> Update a workflow to the latest version
antfarm workflow uninstall <id> Remove a workflow and all associated resources
antfarm workflow logs <task> View logs for a workflow run
